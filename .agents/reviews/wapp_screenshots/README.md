# Web App Review Screenshots

This directory stores screenshots captured during web app/UI audits.

## Naming Convention

Screenshots MUST be named with the date of the review and a descriptive slug:
`YYYY-MM-DD_[tab-or-feature].png`

Examples:
- `2026-05-12_live-deck.png`
- `2026-05-12_topology-live.png`

## How to use in reviews

In the `YYYY-MM-DD_webapp_review.md` file, embed the image using a relative path:

```markdown
![Live Deck](wapp_screenshots/2026-05-12_live-deck.png)
```
